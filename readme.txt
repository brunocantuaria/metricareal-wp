Métrica Real irá adicionar o script de audiência de nosso serviço automaticamente ao seu site.

== Description ==

Para facilitar a instalação de nosso script de audiência, disponibilizamos este plugin que irá instalar automaticamente nosso script em seu site.

O plugin foi pensado para ser plug-and-play, basta instalar e ativar, que ele automaticamente entrará em ação.

O plugin também se integra aos principais plugins de AMP e Web Stories disponíveis para também monitorar essas páginas. Lista de plugins suportados:

*   AMP Oficial
*   AMP for WP
*   AMP WP
*   Web Stories do Google
*   MakeStories

== Installation ==

1. Envie o plugin para seu servidor e descompacte na pasta `/wp-content/plugins/`
2. Ative o plugin através do menu "plugins" dentro do painel do WordPress

== Frequently Asked Questions ==

= O plugin pode deixar meu site lento? =

Não. Além de leve e compatível com todos os navegadores, nem nosso script nem nosso plugin afetam o desempenho do seu site.

= Devo pedir permissão do meu usuário antes de utilizar o plugin? =

Não. Embora recomendamos que avise o usuário que dados de análise de audiência serão enviados para nosso serviço, não é necessário solicitar a autorização visto que os dados coletados são exclusivamente para mensuração de audiência e não são coletados dados sensíveis, conforme disposto pela LGPD.

= Posso utilizar o plugin sem cadastro no Métrica Real? =

Não. Embora você possa instalar e manter o plugin ativo mesmo sem cadastro, qualquer dado enviado aos nossos sistemas será descartado.

= Como cadastro meu site no Métrica Real? =

No momento, apenas sites convidados podem utilizar o serviço do Métrica Real. Os convites são oferecidos por parceiros comerciais do Métrica Real.
